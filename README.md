# Landing Gestionali

## Avvio rapido
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Struttura
- `src/App.tsx`: pagina landing principale (sezioni e copy)
- `src/components/`: componenti UI e sezioni
- `src/index.css`: stile globale + utility custom
- `public/`: asset statici (immagini, logo)

## Note
- Stack: React + TypeScript + Vite + Tailwind
- Avvio locale: http://localhost:5173
